# TIMELINE - Task Manager SPA  
A minimalist, priority-based task manager with drag-and-drop and persistent storage.  

## Features  
✅ Priority tagging (Low/Medium/High)  
✅ LocalStorage persistence  
✅ Drag-and-drop reordering  
✅ Timer integration  
✅ Dark/light mode
